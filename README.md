# Cryptopals Solutions

<img align="right" src="splash.jpg">

Solutions to challenges from https://cryptopals.com/ written in Python 3.

Install and run using `pipenv`:

    pipenv install --python 3.7
    pipenv run ./cryptopals_5.py
